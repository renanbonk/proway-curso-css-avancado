Exemplos de Sombreamento com box-shadow

A propriedade box-shadow é usada paraadicionar sombras aos elementos. permite criar efeitos de profundidade e dar um aspecto masi realista aos elementos da página. A sintaxe é a seguinte:

box shadow: h-offset v-off-set blur spread color

O que seria cada parte?

  • h-offset : Deslocamento horizontal da sombra. valores positivos deslocam a sombra para direita e valores para esquerda
  • v-offset : Deslocamento vertical da sombra. valores positivos deslocam a sombra para baixo e valores para cima
  • blur : Grau de desfoque da sombra. quanto maior o valor, mais desfocada será a sombra. 0 cria uma sombra nítida.
  • spread : Define o quanto a sombra se expande ou contrai. valores positivos expande a sombra e valores negativos contraem.
  • color : Define a cor da sombra

box-shadow aceita mútiplas sombras separadas por vírgula.
