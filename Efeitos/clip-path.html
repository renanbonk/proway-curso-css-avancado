clip-path

O clip-path é uma propriedade CSS que permite criar formas complexas cortamdo elementos de acordo com um caminho especifico. É ultil par criar layouts e efeitos visuais interessantes.

Como funciona o clip-path

O

clip-pathdefine uma area vísivel de um elemento, escondendo o que está fora dessa área. Existem várias formas que você pode usar com clip-path, como:

  • polygon(): Define uma forma usando uma lista de cordenadas que formam um polígono.
  • circle(): Define uma forma circula.
  • ellipse: Define uma forma elíptias.
  • inset(): Defini um retângulo intero com uma margem opcional.
